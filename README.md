## ü§ù Contributing Translations

We welcome contributions to help translate this documentation into more languages! Here's how you can help:

### Adding a New Language

1. **Fork this repository** and create a new branch for your translation
2. **Create a new file** in the `docs/` directory named `README.[language-code].md` (e.g., `README.de.md` for German)
3. **Translate the content** from one of the existing language files (we recommend using `README.en.md` as the source)
4. **Add your language** to the language selection section at the top of this README:
   - Add a badge: `[![de](https://img.shields.io/badge/lang-de-green.svg)](docs/README.de.md)` (replace `de` and `green` with your language code and color)
   - Add a link in the "Available Languages" section
5. **Submit a pull request** with your translation

### Updating Existing Translations

If you notice any errors or outdated information in existing translations:

1. **Fork this repository** and create a new branch
2. **Edit the relevant** `docs/README.[language-code].md` file
3. **Submit a pull request** describing the changes

### Translation Guidelines

- Keep the structure and formatting consistent with other language versions
- Ensure all sections from the original are translated
- Use clear, simple language that's easy to understand
- Maintain the same level of technical detail as the source
- Test all links to ensure they work correctly
