## GitHub Copilot Demonstration

This repository includes a demonstration of GitHub Copilot functionality through `copilot-demo.js`.

### Running the Demo

To invoke and test the Copilot demonstration:

```bash
node copilot-demo.js
```

### Features Demonstrated

- Utility functions with JSDoc comments
- Array operations (sum, average)
- String manipulation (palindrome check, capitalization)
- Random number generation

This demonstrates how GitHub Copilot can assist with writing common programming patterns and well-documented code.
