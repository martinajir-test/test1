# Test1 Repository

A test repository demonstrating GitHub workflows and collaborative development practices.

## Key Features

- **Continuous Integration (CI)**: Automated workflow that triggers on push and pull request events to the main branch
- **PR Approval System**: Built-in pull request approval workflow requiring review before execution
- **GitHub Actions Integration**: Multiple workflows configured for different automation scenarios
- **Test Environment**: Dedicated test environment setup for deployment testing
- **Workflow Dispatch**: Manual workflow execution capability for on-demand testing

## Highlights

- Automated deployment steps with environment protection
- Multi-job workflow setup (deployment and build jobs)
- Branch protection and merge controls
- Easy-to-extend workflow templates
